# Python OOP - Abstract Classes, Interfaces, Duck Typing

Ce dossier contient des exercices sur la programmation orientée objet (OOP) en Python, avec un focus sur :

- Les classes abstraites (Abstract Base Classes)
- Les interfaces et le duck typing
- Le subclassing et la spécialisation de comportements
- L'utilisation de l'héritage multiple et des mixins

## Exercices présents

- **task_00_abc.py** : Classe abstraite `Animal` et ses sous-classes `Dog` et `Cat`.
- **task_01_duck_typing.py** : Interface `Shape`, classes `Circle` et `Rectangle`, et fonction `shape_info` utilisant le duck typing.

Chaque exercice est accompagné d'un fichier de test exécutable pour valider le comportement attendu.

## Exécution des tests

```bash
./main_00_abc.py
./main_01_duck_typing.py
```
